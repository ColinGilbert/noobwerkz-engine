Detailed Description

template<class MeshItems, class Connectivity>
class OpenMesh::AttribKernelT< MeshItems, Connectivity >

The attribute kernel adds all standard properties to the kernel.

Therefore the functions/types defined here provide a subset of the kernel interface as described in Concepts::KernelT.

See also
Concepts::KernelT

Assignment from another mesh of another type.

Note
